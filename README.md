# 🛒 Mini E-Commerce API

A scalable Node.js backend for an e-commerce system featuring cart management, checkout flow, and advanced features like authentication, order tracking, and product management.

---

## 🚀 Features

### ✅ v1

* Add to cart
* Remove from cart
* Update cart items
* Basic checkout flow
* Cart total calculation

---

### 🔥 v2 (In Progress)

* Authorization (Admin/User roles)
* Product pagination & search
* Cart improvements
* Order history
* Order status tracking
* Product image upload
* Soft delete system

---

## 🧱 Tech Stack

* Node.js
* Express.js
* MySQL (Sequelize ORM)

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Run the server

```bash
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file:

```
DB_HOST=
DB_USER=
DB_PASS=
DB_NAME=
PORT=
JWT_SECRET=
```
